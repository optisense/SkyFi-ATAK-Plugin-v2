## Installation
1. Download the APK file above
2. Enable "Unknown Sources" in Android settings
3. Install the APK
4. Open ATAK 5.4.0
5. Access via SkyFi toolbar icon

## Requirements
- ATAK 5.4.0 CIV
- Android 5.0+ (API 21+)
- Active SkyFi account
