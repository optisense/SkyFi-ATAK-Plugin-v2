SkyFi ATAK Plugin v2.0-beta1 Bundle
===================================

This bundle contains:
1. atak-civ-5.4.0.18.apk - ATAK CIV version 5.4.0 (required)
2. skyfi-plugin-v2.0-beta1-civ-debug.apk - SkyFi Plugin v2.0-beta1

INSTALLATION INSTRUCTIONS:
-------------------------

1. Install ATAK first:
   adb install atak-civ-5.4.0.18.apk

2. Install the SkyFi plugin:
   adb install -r skyfi-plugin-v2.0-beta1-civ-debug.apk

3. Launch ATAK

4. Enable the plugin:
   - Go to Settings → Tool Preferences → Plugin Management
   - Find "SkyFi" and toggle it ON
   - Restart ATAK if prompted

NEW FEATURES IN v2.0-beta1:
---------------------------
- MGRS coordinate support
- Image cache/archive/favorite buttons
- Assured tasking with pricing
- Bug fixes and stability improvements

IMPORTANT:
----------
- This plugin requires ATAK 5.4.0 (NOT the Play Store version)
- This is a beta release for Space Force testing
- Report issues to the development team
